# About Us

JakMods.dev is maintained by barg034 and zedb0t. 

Back in 2022 we got started with OpenGOAL mods working on the [Checkpoint Randomizer](https://opengoal.dev/docs/developing/modding_examples/checkpoint_randomizer). To make it easier to distribute updates and new mods to players, we built the [mod launcher](/docs/legacy) as a small Python app to help manage installing and updating mods.

Over time mods grew in number and popularity, and we were increasingly contributing more and more back to the upstream OpenGOAL project. In 2024 we helped add mod support to the vanilla OpenGOAL launcher, and officially deprecated the separate mod launcher.

Besides working on our own mods, we try to help out with bugfixes, tooling enhancements, and documentation around the project. We also manage a Discord server where we can offer guidance and support for mod developers.

We hope you enjoy the Jak Mods that we and others have created!
